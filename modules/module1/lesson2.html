Урок 2: Основи HTML – будівельні блоки веб-сторінки

Що таке HTML? Теги, елементи, атрибути

HTML (HyperText Markup Language) – це не мова програмування, а мова розмітки. Вона використовується для структурування веб-сторінок та їх вмісту, визначаючи, що є заголовком, параграфом, посиланням, зображенням тощо.

Теги та Елементи

HTML-документи складаються з HTML-елементів. Більшість елементів складаються з:

  • Початкового тега (Opening Tag): Наприклад, <p>. Він вказує на початок елемента.
  • Кінцевого тега (Closing Tag): Наприклад, </p>. Він вказує на кінець елемента.
  • Вмісту (Content): Все, що знаходиться між початковим та кінцевим тегами.

Разом вони утворюють **елемент**:

<p>Це є елемент параграфа.</p>

Деякі елементи є порожніми (empty elements), тобто вони не мають вмісту і кінцевого тега. Наприклад:

  • <img> (для зображень)
  • <br> (для розриву рядка)
  • <hr> (для горизонтальної лінії)

Атрибути

Атрибути надають додаткову інформацію про HTML-елементи. Вони завжди вказуються у початковому тегу і складаються з пари "назва="значення"".

<a href="https://example.com">Це посилання</a>

Тут href – це назва атрибута, а "https://example.com" – його значення.

Важливо: Атрибути завжди мають бути в лапках (одинарних або подвійних) і не чутливі до регістру, хоча для кращої практики рекомендується використовувати нижній регістр.

Базова структура HTML-документа (HTML5)

Кожен HTML5-документ повинен мати таку базову структуру:

<!DOCTYPE html>
<html lang="uk">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Назва вашої сторінки</title>
    <meta name="description" content="Короткий опис сторінки для пошукових систем.">
    <meta name="keywords" content="ключові, слова, через, кому">
    <link rel="stylesheet" href="css/style.css"> <!-- Підключення CSS файлу -->
    <script src="js/script.js"></script> <!-- Підключення JavaScript файлу -->
</head>
<body>
    <!-- Весь видимий вміст вашої веб-сторінки тут -->
</body>
</html>
  • <!DOCTYPE html>: Декларація типу документа. Вказує браузеру, що це HTML5-документ. Завжди на самому початку!
  • <html lang="uk">: Кореневий елемент HTML-сторінки. Атрибут lang="uk" вказує мову контенту сторінки, що важливо для доступності та SEO.
  • <head>: Містить мета-інформацію про сторінку, яка не відображається безпосередньо у браузері, але є важливою для нього та пошукових систем.
    • <meta charset="UTF-8">: Вказує кодування символів для коректного відображення тексту (особливо українських символів).
    • <meta name="viewport" content="width=device-width, initial-scale=1.0">: Дуже важливий для адаптивного дизайну. Вказує браузеру, як масштабувати сторінку на різних пристроях.
    • <title>: Визначає заголовок сторінки, який відображається у вкладці браузера або вікні.
    • <meta name="description"> та <meta name="keywords">: Надають опис та ключові слова для пошукових систем.
    • <link rel="stylesheet" href="css/style.css">: Підключає зовнішній файл CSS.
    • <script src="js/script.js"></script>: Підключає зовнішній файл JavaScript. Зазвичай розміщується в кінці <body> для кращої продуктивності.
  • <body>: Містить весь видимий контент веб-сторінки: текст, зображення, посилання, таблиці, форми тощо.

Заголовки та параграфи

HTML надає 6 рівнів заголовків:

<h1>Це Заголовок Рівня 1</h1>
<h2>Це Заголовок Рівня 2</h2>
<h3>Це Заголовок Рівня 3</h3>
<h4>Це Заголовок Рівня 4</h4>
<h5>Це Заголовок Рівня 5</h5>
<h6>Це Заголовок Рівня 6</h6>

Використовуйте заголовки для створення ієрархії та структури контенту. <h1> повинен бути лише один на сторінку і представляти головний заголовок.

Параграфи створюються за допомогою тега <p>:

<p>Це перший параграф тексту на моїй веб-сторінці.</p>
<p>А це вже другий параграф.</p>

Для розриву рядка (переходу на новий рядок без створення нового параграфа) використовуйте тег <br>:

<p>Це перший рядок.<br>Це вже другий рядок в тому ж параграфі.</p>

Для створення горизонтальної розділювальної лінії використовуйте тег <hr>:

<p>Текст до розділювача.</p>
<hr>
<p>Текст після розділювача.</p>

Форматування тексту (наприклад, жирний, курсив)

Для логічного виділення тексту (що також важливо для скрінрідерів та SEO):

  • <strong>: Визначає текст з сильним значенням (зазвичай відображається жирним).
  • <em>: Визначає текст з акцентом (зазвичай відображається курсивом).

Приклад:

<p>Це <strong>дуже важливий</strong> текст, а це <em>акцентована</em> фраза.</p>

Примітка: Хоча є також теги <b> (жирний) та <i> (курсив), вони використовуються для візуального форматування без надання додаткового семантичного значення. Для кращої практики та доступності, <strong> та <em> є кращим вибором.

Посилання (Links)

Посилання (гіперпосилання) дозволяють користувачам переходити між веб-сторінками або до різних частин однієї сторінки. Створюються за допомогою тега <a> (anchor).

<a href="https://google.com">Відвідати Google</a>
  • href (Hypertext Reference): Обов'язковий атрибут, що вказує URL-адресу, на яку посилається гіперпосилання.
  • target: Атрибут, що вказує, де відкривати пов'язаний документ.
    • _self (за замовчуванням): Відкриває у поточному вікні/вкладці.
    • _blank: Відкриває у новому вікні або вкладці.
    • _parent: Відкриває у батьківському фреймі.
    • _top: Відкриває у повній робочій області вікна.

Приклад з target="_blank":

<p>Більше інформації на <a href="https://uk.wikipedia.org" target="_blank">Wikipedia</a>.</p>

Відносні та абсолютні посилання

  • Абсолютні посилання: Містять повний URL, включаючи протокол та домен (наприклад, https://www.example.com/page.html). Використовуються для посилань на зовнішні ресурси.
  • Відносні посилання: Вказують шлях до файлу відносно поточної сторінки. Використовуються для посилань на сторінки в межах того ж веб-сайту.
    • <a href="about.html">Про нас</a> (файл about.html знаходиться в тій же папці)
    • <a href="pages/contact.html">Контакти</a> (файл contact.html знаходиться у підпапці pages)
    • <a href="../index.html">На головну</a> (файл index.html знаходиться в папці на один рівень вище)

Зображення (Images)

Зображення вбудовуються за допомогою порожнього тега <img>.

<img src="images/my_image.jpg" alt="Опис мого зображення" width="300" height="200">
  • src (Source): Обов'язковий атрибут, що вказує шлях до файлу зображення. Може бути абсолютним або відносним.
  • alt (Alternative Text): Дуже важливий атрибут, який надає альтернативний текстовий опис зображення. Цей текст відображається, якщо зображення не завантажилося, і читається скрінрідерами для користувачів з обмеженими можливостями. Критично важливий для доступності та SEO.
  • width та height: Вказують ширину та висоту зображення в пікселях. Рекомендується задавати їх для уникнення "стрибків" макета під час завантаження сторінки. Однак, краще контролювати розмір зображень через CSS.

Списки (Lists)

HTML підтримує два основні типи списків:

Невпорядковані списки (Unordered Lists)

Елементи позначаються маркерами (зазвичай кружечками). Створюються за допомогою тега <ul> (unordered list), кожен елемент списку – <li> (list item).

<ul>
    <li>Яблука</li>
    <li>Апельсини</li>
    <li>Банани</li>
</ul>

Впорядковані списки (Ordered Lists)

Елементи позначаються числами, літерами або римськими цифрами. Створюються за допомогою тега <ol> (ordered list), кожен елемент списку – <li>.

<ol>
    <li>Перший крок</li>
    <li>Другий крок</li>
    <li>Третій крок</li>
</ol>

Ви можете вкладати списки один в одного для створення багаторівневих списків.

Практичне завдання: Створення веб-сторінки "Мій улюблений рецепт"

Створіть новий HTML-файл my_recipe.html і реалізуйте наступне:

  1. Налаштуйте базову структуру HTML5 (!DOCTYPE, html, head, body).
  2. Задайте заголовок сторінки (<title>) на кшталт "Мій улюблений рецепт [назва страви]".
  3. У <body>:
    • Створіть головний заголовок <h1> з назвою рецепта.
    • Додайте короткий опис страви за допомогою <p>.
    • Вставте зображення готової страви (можна знайти в інтернеті, не забудьте про атрибути src та alt). Задайте йому розміри, наприклад, width="400".
    • Створіть заголовок <h2> "Інгредієнти".
    • Перерахуйте інгредієнти за допомогою невпорядкованого списку (<ul> та <li>).
    • Створіть заголовок <h2> "Інструкції приготування".
    • Опишіть кроки приготування за допомогою впорядкованого списку (<ol> та <li>).
    • Додайте в кінці параграф <p> з фразою на кшталт "Смачного! Більше рецептів на <a href="#">нашому сайті</a>." (посилання поки може вести на `#`).
    • Використайте теги <strong> та <em> для виділення ключових слів в описі або інструкціях.

Збережіть файл і відкрийте його у браузері, щоб перевірити, як виглядає ваша сторінка.
